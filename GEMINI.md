When the repository open you do not load any file in it, but load the files open in the editor.

In this repository I am working on building an multi model agentic AI system that works on generating a description for a code file, directory of code files and scripts, or a full repository, then conduct a detailed assessment for the code correctness in functionality and generate a score in percentage and another assessment for the code style and generate a score in percentage, then suggest a few possible corrections or improvements, and write all of
that in a report.
